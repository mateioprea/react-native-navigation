## `visible`

| Type   | Required | Description                                                                                                                                                    |
| ------ | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| boolean | No       | Determines if SearchBar is visible or not |

## `hideOnScroll`

| Type                    | Required | Description                    |
| ----------------------- | -------- | ------------------------------ |
| boolean | No       | Hides the UISearchBar when scrolling. |

## `hideTopBarOnFocus`

| Type               | Required | Description                   |
| ------------------ | -------- | ----------------------------- |
| boolean | No      | Indicates whether the navigation bar should be hidden when searching. True by default. |

## `obscuresBackgroundDuringPresentation`

| Type                    | Required | Description                    |
| ----------------------- | -------- | ------------------------------ |
| boolean | No       | A Boolean indicating whether the underlying content is obscured during a search. |

## `backgroundColor`

| Type                    | Required | Description                    |
| ----------------------- | -------- | ------------------------------ |
| Color | No       | The background color of the UISearchBar's TextField. |

## `tintColor`

| Type                    | Required | Description                    |
| ----------------------- | -------- | ------------------------------ |
| Color | No       | The tint color of the UISearchBar. Affects text selection color, as well as "Cancel" button color. |

## `searchBarPlaceholder`

| Type                    | Required | Description                    |
| ----------------------- | -------- | ------------------------------ |
| string | No       | The placeholder value in the UISearchBar. |
